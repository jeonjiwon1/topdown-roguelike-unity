Drone 시스템 확장

Drone Mode 시스템

	1 : Heal Mode
	2 : Buff Mode
	3 : Attack Mode
	키 입력으로 드론 모드 전환 가능
구현 스크립트
	DroneMode.cs
	DroneController.cs

	일정 간격마다 투사체 발사
	Attack Mode일 때만 동작하도록 구현
구현 스크립트
	DroneAttackModule.cs

Drone Buff Module

	PlayerWeaponController에 Damage Multiplier 구조 추가
	드론 Buff Mode에서 플레이어 공격력 상승 적용
구현 스크립트
	DroneBuffModule.cs

전투 시스템 설계 확장

게임 진행 구조 설계

Chapter (행성 테마)
	행성별 테마 Enemy
	챕터별 적 능력치 상승

Stage (층 구조)
	행성 내 진행 층
	층 진행에 따른 난이도 상승

Wave (전투 단위)
	적 수 증가
	적 조합 변화
	전투 압박 증가


전투 설계 방향 결정

웨이브 난이도 상승 방식
	웨이브 자체에서 적 능력치를 크게 상승시키지 않음
	적 수 / 조합 / 전투 패턴 중심으로 난이도 조절

난이도 상승 구조

	Chapter → 적 기본 능력치 상승
	Stage → 점진적 난이도 상승
	Wave → 전투 패턴 변화

상점 / 증강 / 정비 구간 이후
	다음 구간 적 난이도 상승 구조 설계


추가 전투 시스템 설계

Spawn Indicator 시스템 설계
	적 등장 위치 사전 표시
	0.5 ~ 0.7초 후 Enemy 스폰
	Burst 스폰 시 공정성 확보 목적

Enemy Director 시스템 검토
	플레이어 상태 기반 적 스폰 밀도 조절
	전투 긴장도 자동 조절 시스템
